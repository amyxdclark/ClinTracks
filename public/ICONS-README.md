# PWA Icons

For production deployment, replace these placeholder files with proper icon images:

- `pwa-192x192.png` - 192x192 pixel icon
- `pwa-512x512.png` - 512x512 pixel icon
- `apple-touch-icon.png` - 180x180 pixel icon for iOS
- `favicon.ico` - Standard favicon

For now, the app will use the default Vite icon as a fallback.
